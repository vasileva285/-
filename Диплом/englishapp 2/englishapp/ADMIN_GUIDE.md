# Руководство по админ-панели EnglishApp

## 🔐 Доступ к админ-панели

Админ-панель доступна только для пользователей с правами администратора (staff).

**URL:** `http://127.0.0.1:8000/admin/`

## 👤 Создание администратора

### Способ 1: Через команду Django

```bash
cd /Users/daniilakkerman/Documents/englishapp
source venv/bin/activate
python manage.py createsuperuser
```

Введите:
- Username (имя пользователя)
- Email (опционально)
- Password (пароль)
- Подтверждение пароля

### Способ 2: Через Django shell

```bash
python manage.py shell
```

```python
from django.contrib.auth.models import User
user = User.objects.create_user('admin', 'admin@example.com', 'password')
user.is_staff = True
user.is_superuser = True
user.save()
```

## 📚 Управление контентом

### 1. Категории уроков (Lesson Categories)

**Путь:** Админ-панель → Категории уроков

**Что можно делать:**
- ✅ Добавлять новые категории (Present, Past, Future и т.д.)
- ✅ Редактировать существующие категории
- ✅ Устанавливать порядок отображения
- ✅ Добавлять иконки (эмодзи)
- ✅ Просматривать количество уроков в категории

**Поля:**
- **Название** - название категории
- **URL** - автоматически генерируется из названия
- **Описание** - описание категории
- **Иконка** - эмодзи для отображения (например: ⏰, 📜, 🔮)
- **Порядок** - порядок сортировки

### 2. Уроки (Lessons)

**Путь:** Админ-панель → Уроки

**Что можно делать:**
- ✅ Создавать новые уроки
- ✅ Добавлять теоретический материал
- ✅ Добавлять примеры использования
- ✅ Добавлять упражнения прямо в урок (inline)
- ✅ Устанавливать порядок уроков
- ✅ Активировать/деактивировать уроки

**Поля:**
- **Категория** - выбор категории урока
- **Название** - название урока
- **URL** - автоматически генерируется
- **Описание** - краткое описание
- **Теоретический материал** - основной текст урока
- **Примеры** - примеры использования (разделять новой строкой)
- **Порядок** - порядок в категории
- **Активен** - включить/выключить урок

**Особенности:**
- Можно добавлять упражнения прямо при создании урока
- Просмотр количества упражнений в уроке
- Поиск по названию и описанию

### 3. Упражнения (Exercises)

**Путь:** Админ-панель → Упражнения

**Что можно делать:**
- ✅ Создавать упражнения различных типов
- ✅ Устанавливать баллы за упражнение
- ✅ Добавлять объяснения правильных ответов
- ✅ Настраивать варианты ответов для множественного выбора

**Типы упражнений:**

#### a) Множественный выбор (Multiple Choice)
- **Вопрос** - текст вопроса
- **Правильный ответ** - правильный вариант (текст)
- **Варианты ответов** - JSON массив: `["Вариант 1", "Вариант 2", "Вариант 3"]`
- **Объяснение** - объяснение правильного ответа
- **Баллы** - количество баллов за правильный ответ

**Пример вариантов ответов:**
```json
["go", "goes", "going", "went"]
```

#### b) Заполнение пропусков (Fill Blank)
- **Вопрос** - текст с пропуском: "She ___ (work) in a hospital."
- **Правильный ответ** - правильный ответ: "works"
- **Объяснение** - объяснение

#### c) Перевод (Translation)
- **Вопрос** - текст для перевода: "Переведите на английский: 'Я работаю каждый день'"
- **Правильный ответ** - можно указать несколько вариантов через `|`: 
  `"I work every day|I work daily"`
- **Объяснение** - объяснение

#### d) Вопрос (Question)
- **Вопрос** - вопрос
- **Правильный ответ** - правильный ответ
- **Объяснение** - объяснение

**Поля:**
- **Урок** - к какому уроку относится упражнение
- **Тип упражнения** - выбор типа
- **Вопрос/Задание** - текст задания
- **Правильный ответ** - правильный ответ
- **Варианты ответов** - для множественного выбора (JSON)
- **Объяснение** - объяснение правильного ответа
- **Порядок** - порядок в уроке
- **Баллы** - количество баллов (1-100)

### 4. Прогресс пользователей (User Progress)

**Путь:** Админ-панель → Прогресс пользователей

**Что можно делать:**
- ✅ Просматривать прогресс всех пользователей
- ✅ Видеть завершенные уроки
- ✅ Просматривать баллы пользователей
- ✅ Фильтровать по урокам и статусу завершения

**Информация:**
- Пользователь
- Урок
- Статус завершения
- Баллы (заработано / всего)
- Процент выполнения
- Дата начала
- Дата завершения

### 5. Попытки выполнения упражнений (User Exercise Attempts)

**Путь:** Админ-панель → Попытки выполнения упражнений

**Что можно делать:**
- ✅ Просматривать все попытки пользователей
- ✅ Видеть правильные и неправильные ответы
- ✅ Анализировать статистику выполнения

**Информация:**
- Пользователь
- Упражнение
- Ответ пользователя
- Правильность ответа
- Заработанные баллы
- Дата попытки

## 🎯 Быстрый старт: Создание урока

### Шаг 1: Создайте категорию (если нужно)
1. Перейдите в "Категории уроков"
2. Нажмите "Добавить категорию уроков"
3. Заполните поля и сохраните

### Шаг 2: Создайте урок
1. Перейдите в "Уроки"
2. Нажмите "Добавить урок"
3. Заполните:
   - Категория
   - Название
   - Описание
   - Теоретический материал (основной текст)
   - Примеры (каждый с новой строки)
   - Порядок
   - Активен ✓
4. Внизу страницы добавьте упражнения (можно добавить несколько)
5. Сохраните

### Шаг 3: Добавьте упражнения
При создании урока можно добавить упражнения inline, или:
1. Перейдите в "Упражнения"
2. Нажмите "Добавить упражнение"
3. Выберите урок
4. Выберите тип упражнения
5. Заполните все поля
6. Для множественного выбора укажите варианты в JSON формате
7. Сохраните

## 💡 Советы

1. **Порядок важен** - устанавливайте порядок для категорий и уроков, чтобы они отображались в нужной последовательности

2. **Активность** - деактивируйте уроки, которые еще не готовы, чтобы пользователи их не видели

3. **Варианты ответов** - для множественного выбора всегда включайте правильный ответ в список вариантов

4. **Объяснения** - добавляйте объяснения к упражнениям, это поможет пользователям лучше понять материал

5. **Баллы** - устанавливайте разное количество баллов в зависимости от сложности упражнения

6. **Переводы** - для упражнений на перевод используйте символ `|` для указания нескольких правильных вариантов

## 🔒 Безопасность

- Админ-панель доступна только пользователям с `is_staff = True`
- Используйте сильные пароли для администраторов
- Не делитесь учетными данными администратора

## 📊 Статистика

- Количество упражнений в каждом уроке
- Прогресс всех пользователей
- Попытки выполнения упражнений
