# Макеты страниц EnglishApp

## 📐 Файл с макетами

Файл `page_mockups.drawio` содержит визуальные макеты всех основных страниц приложения.

## 🎨 Описание макетов

### 1. Главная страница (Home)
- **Hero секция** - приветственный блок с градиентом
- **Статистика** - 3 карточки с показателями (уроки, баллы, активность)
- **Категории** - карточки категорий с прогресс-барами и кнопками

### 2. Страница категории (Category Detail)
- **Заголовок категории** - большой блок с иконкой и описанием
- **Список уроков** - карточки уроков с прогрессом и статусом
- **Индикаторы** - ✓ Завершен, ⏳ В процессе

### 3. Страница урока (Lesson Detail)
- **Теоретический материал** - блок с теорией
- **Примеры** - блок с примерами использования
- **Упражнения** - список упражнений с статусами
- **Боковая панель** - прогресс урока

### 4. Страница упражнения (Exercise Detail)
- **Карточка упражнения** - вопрос и варианты ответов
- **Тип упражнения** - множественный выбор, заполнение пропусков и т.д.
- **Результат** - уведомление о правильности ответа

### 5. Страница прогресса (Progress)
- **Общая статистика** - уроки, баллы, точность
- **Прогресс по категориям** - визуализация прогресса

### 6. Настройки профиля (Settings)
- **Статистика пользователя** - краткая информация
- **Форма редактирования** - поля для изменения данных

### 7. Регистрация / Вход (Auth)
- **Форма регистрации** - все необходимые поля
- **Форма входа** - логин и пароль

## 🎨 Цветовая схема

- **Primary (Синий)**: #0d6efd - основные элементы, кнопки
- **Success (Зеленый)**: #198754 - завершенные уроки, успех
- **Warning (Желтый)**: #ffc107 - в процессе, предупреждения
- **Danger (Красный)**: #dc3545 - ошибки, не начато
- **Info (Голубой)**: #0dcaf0 - информация, категории
- **Gradient**: #667eea → #764ba2 - hero секции

## 📱 Адаптивность

Все макеты адаптивны и работают на:
- 💻 Десктоп (1200px+)
- 📱 Планшет (768px - 1199px)
- 📱 Мобильный (< 768px)

## 🔧 Как открыть макеты

1. Откройте [draw.io](https://app.diagrams.net/) в браузере
2. Файл → Открыть → Выберите `page_mockups.drawio`
3. Или перетащите файл в окно draw.io

## 📋 Элементы дизайна

### Карточки
- Скругленные углы (rounded=1)
- Тени для глубины
- Цветные границы для категоризации

### Кнопки
- Скругленные углы
- Hover эффекты (при наведении)
- Иконки для визуального понимания

### Прогресс-бары
- Визуальное представление прогресса
- Цветовая индикация (зеленый/желтый/красный)
- Процентное отображение

### Иконки
- 📚 Книга - уроки
- ⭐ Звезда - баллы
- 🔥 Огонь - активность
- ⏰ Часы - Present Tense
- 📜 Свиток - Past Tense
- 🔮 Кристалл - Future Tense
- ✓ Галочка - завершено
- ⏳ Песочные часы - в процессе
