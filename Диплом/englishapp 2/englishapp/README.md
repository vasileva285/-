# EnglishApp - Приложение для изучения английского языка

Приложение для изучения английского языка в стиле Duolingo и Busuu, построенное на Django.

## Функциональность

### Функциональные требования:
- ✅ Регистрация и авторизация пользователей
- ✅ Выбор уроков по категориям (Present, Past, Future)
- ✅ Интерактивные упражнения (множественный выбор, заполнение пропусков, перевод, вопросы)
- ✅ Теоретический материал с примерами и объяснениями
- ✅ Система тестирования с автоматической проверкой и оцениванием
- ✅ Отслеживание прогресса пользователя и возврат к предыдущим занятиям

### Нефункциональные требования:
- ✅ Удобный и интуитивно понятный интерфейс
- ✅ Быстрая загрузка страниц
- ✅ Адаптивный дизайн для всех устройств (ПК, планшеты, смартфоны)
- ✅ Оптимизация для работы при низкой скорости интернета

## Установка и запуск

### Требования
- Python 3.8+
- Django 4.2+

### Шаги установки

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Примените миграции:**
```bash
python manage.py migrate
```

3. **Создайте суперпользователя (опционально):**
```bash
python manage.py createsuperuser
```

4. **Загрузите начальные данные (категории, уроки, упражнения):**
```bash
python manage.py load_initial_data
```

5. **Запустите сервер разработки:**
```bash
python manage.py runserver
```

6. **Откройте браузер и перейдите по адресу:**
```
http://127.0.0.1:8000
```

## Использование

### Для пользователей:

1. **Регистрация:** Создайте аккаунт на странице регистрации
2. **Выбор категории:** Выберите категорию уроков (Present, Past, Future)
3. **Изучение теории:** Прочитайте теоретический материал урока
4. **Выполнение упражнений:** Выполните интерактивные упражнения
5. **Отслеживание прогресса:** Просматривайте свой прогресс на странице "Прогресс"

### Для администраторов:

1. Войдите в админ-панель: `http://127.0.0.1:8000/admin/`
2. Управляйте категориями, уроками и упражнениями
3. Просматривайте статистику пользователей

## Типы упражнений

- **Множественный выбор (Multiple Choice)** - выбор правильного ответа из предложенных вариантов
- **Заполнение пропусков (Fill Blank)** - ввод правильного ответа в текстовое поле
- **Перевод (Translation)** - перевод предложения на английский язык
- **Вопрос (Question)** - ответ на вопрос

## Модели данных

- **LessonCategory** - категории уроков (Present, Past, Future)
- **Lesson** - уроки с теорией и примерами
- **Exercise** - упражнения различных типов
- **UserProgress** - прогресс пользователя по урокам
- **UserExerciseAttempt** - попытки выполнения упражнений

## Технологии

- **Backend:** Django 4.2
- **Frontend:** Bootstrap 5, Bootstrap Icons
- **Database:** SQLite (по умолчанию)
- **Forms:** Django Crispy Forms

## Разработка

### Добавление новых уроков

1. Войдите в админ-панель
2. Перейдите в раздел "Уроки"
3. Создайте новый урок, выбрав категорию
4. Добавьте теоретический материал и примеры
5. Создайте упражнения для урока

### Добавление новых упражнений

1. В админ-панели перейдите в раздел "Упражнения"
2. Выберите урок
3. Выберите тип упражнения
4. Введите вопрос и правильный ответ
5. Для множественного выбора добавьте варианты в поле "Варианты ответов" (JSON формат)
