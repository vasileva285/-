# Быстрый старт

## 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

## 2. Применение миграций
```bash
python manage.py migrate
```

## 3. Загрузка начальных данных
```bash
python manage.py load_initial_data
```

Эта команда создаст:
- 3 категории уроков (Present, Past, Future)
- 4 урока с теорией и примерами
- 10+ упражнений различных типов

## 4. Создание администратора (обязательно для управления контентом)
```bash
# Способ 1: Через Django команду
python manage.py createsuperuser

# Способ 2: Через скрипт
python create_admin.py
```

**Важно:** Админ-панель доступна только пользователям с правами администратора!

## 5. Запуск сервера
```bash
python manage.py runserver
```

## 6. Откройте в браузере
```
http://127.0.0.1:8000
```

## Что дальше?

1. **Зарегистрируйтесь** - создайте новый аккаунт
2. **Выберите категорию** - начните с Present Tense
3. **Изучите теорию** - прочитайте материал урока
4. **Выполните упражнения** - практикуйтесь с интерактивными заданиями
5. **Отслеживайте прогресс** - смотрите свою статистику

## Админ-панель

Для доступа к админ-панели:
```
http://127.0.0.1:8000/admin/
```

Здесь вы можете:
- Добавлять новые категории и уроки
- Создавать упражнения
- Просматривать статистику пользователей
