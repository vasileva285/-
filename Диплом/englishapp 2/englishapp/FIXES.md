# Исправленные ошибки

## ✅ Исправлено

### 1. Математические операции в шаблонах Django
**Проблема:** Django шаблоны не поддерживают прямые математические операции типа `|mul:100|div:total`.

**Исправление:**
- Все вычисления процентов перенесены в views
- В шаблонах используются готовые значения `percentage` и `progress_percentage`
- Исправлены файлы:
  - `templates/lessons/home.html`
  - `templates/lessons/category_detail.html`
  - `templates/lessons/lesson_detail.html`

### 2. Вычисление процентов в views
**Добавлено:**
- В `home()`: вычисление `category.percentage`
- В `category_detail()`: вычисление `lesson.progress_percentage`
- В `lesson_detail()`: вычисление `progress.percentage`

### 3. Обновление requirements.txt
**Проблема:** Pillow 10.1.0 не совместим с Python 3.13

**Исправление:** Обновлено до `Pillow>=10.2.0` (установилась версия 12.0.0)

### 4. Предотвращение дублирования баллов
**Проблема:** При повторных правильных ответах баллы начислялись повторно

**Исправление:** Добавлена проверка на существование правильного ответа перед начислением баллов
